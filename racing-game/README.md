# 极速跑车 · 超级赛车 🏎️

一个专为小朋友（约 10 岁）打造的俯视视角赛车游戏，画面精美、操作简单、即开即玩。
纯原生 **HTML5 Canvas + JavaScript**，无任何外部依赖，双击 `index.html` 即可运行。

## ✨ 特色

- **5 辆顶级跑车**，各有速度 / 加速 / 操控特性：
  - 🔴 红色闪电（极速之王）
  - 🟢 青牛猛兽（全能战士）
  - 🔵 蓝色猎鹰（弯道大师）
  - ⚫ 暗夜疾风（直线狂魔）
  - ⚪ 白银飞箭（起步飞人）
- **5 条主题赛道**，画面各具风格：
  - 🌱 阳光草原（新手推荐）
  - 🌊 碧海海岸（起伏弯道）
  - ⛰️ 云顶山道（技术挑战）
  - 🌆 霓虹夜城（炫酷夜跑）
  - 🏜️ 烈日沙漠（高速长弯）
- 3 名电脑对手同场竞速，比拼名次 🏆
- 3 圈计时赛、迷你地图、名次实时显示、最快圈速纪录（本地保存）
- 3、2、1、GO！倒计时、轮胎烟尘、加速喷火、路缘石、格子起跑线等精美细节
- 引擎音效（可一键静音）

## 🎮 操作方式

**键盘**

| 按键 | 功能 |
| --- | --- |
| <kbd>↑</kbd> / <kbd>W</kbd> | 油门 |
| <kbd>↓</kbd> / <kbd>S</kbd> | 刹车 / 倒车 |
| <kbd>←</kbd> / <kbd>A</kbd> | 向左转 |
| <kbd>→</kbd> / <kbd>D</kbd> | 向右转 |

**手机 / 平板**：使用屏幕下方的方向与油门 / 刹车按钮。

## 💡 小提示

- 跑到草地 / 沙地上速度会明显变慢，尽量待在赛道上。
- 过弯前稍微松一点油门，会更容易转弯。
- 选操控高的车（如蓝色猎鹰）过弯更稳，选极速高的车（如暗夜疾风）直道更快。

## 🗂️ 文件结构

- `index.html` — 页面结构与界面元素
- `styles.css` — 界面与 HUD 样式
- `game.js` — 游戏逻辑（赛道生成、物理、AI、渲染、音效）

## 🛠️ 技术栈

- HTML5 Canvas 2D
- 原生 JavaScript（Catmull-Rom 样条生成赛道、俯视相机、粒子特效、Web Audio 音效）
- 无第三方依赖
